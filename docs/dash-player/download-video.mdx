# 下载在线视频

目前 DashPlayer 只支持本地视频文件，但市面上有很多好用的视频下载工具，你可以通过它们将视频下载下来。

-   Windows 平台：[Internet Download Manager (IDM)](https://www.internetdownloadmanager.com/)
-   macOS 平台：[Downie](https://software.charliemonroe.net/downie/)
